## E-commerce project

Simulates simple E-shop

uses Node.js + Express + React + Redux

Features:
* redux usage
* own API/side API
* sorting
* currencies
* actions logging
* local storage usage

To do:

## Quick start
To install dependencies and clean the git repo run:
  $ yarn install
  or
  $ npm install

To run in dev mode:
  $ yarn run dev
    or
  $ npm run dev
